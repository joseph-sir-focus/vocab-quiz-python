# 🧠 Vocabulary Quiz Game

A terminal-based quiz application that presents definitions and challenges users to guess the correct programming terms.

## 🚀 Features
* Randomized Questioning: Uses the `random` library to pick words dynamically.
* Data Validation: Seamlessly handles user inputs case-insensitively using `.lower()`.
* Score Tracking: Tracks correct answers and total questions played dynamically.

## 🛠️ Built With
* Python 3 (Dictionaries, Lists, Randomization Loops)
* Git & GitHub
